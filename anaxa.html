Character Summary

Anaxa is a Skill-based DPS with an focus on weakness implants to enable the rest of his kit. After a certain number of weakness implants on an emey, he can trigger an additional Skill after attacking that enemy. His Ultimate also implants all weakness types to all enemies, which enables his kit faster. While he is listed as an Erudition character, his Skill is a bounce attack, which means that he excels more in single-target situations rather than high-target fights.

Anaxa can also function as a sub-DPS with The Herta, and Anaxa is Herta's best option at lower targets, but since Anaxa does not synergise too much with The Herta in the first place and that Herta is not that great into lower target fights, Anaxa might be better off as his standalone DPS for those fights instead.

Build Notes

Recommended stats here are for a hypercarry Anaxa team and assumes ATK boots + Rutilant Arena + Genius 4pc. Aim for 100% CR if you are using Scholar 4pc.

CR/CD: If you are using Sunday, aim for 80% CR/130% CD instead. The lower CD requirement is because you need 5-6 SPD substats to get to 133.34+ SPD.

SPD: If you are using Cyrene/Cerydra, aim for 113.34+ SPD or base speed (99). 99 SPD is preferred if you are using DDD on Cerydra since you only need 1 DDD activation to get Anaxa to 2 actions in 150 AV. If you are using Cerydra/Sunday, aim for 113.34+ SPD, and with just Sunday, 133.34 SPD. For sub-DPS Anaxa, run SPD boots and get him to 143.9 SPD.

◈ Recommended Light Cones / Relic Sets

Light Cones

Life Should Be Cast to Flames BiS
His signature LC for both main DPS and sub-DPS Anaxa, and complements well with additional DEF shred sources such as Genius 4pc or Cerydra E1. Without Cyrene or Cerydra, this also facilitates easy 2-turn ult rotations for Anaxa.
Into the Unreachable Veil Alternative
Herta's signature LC and a premium option for Anaxa; this LC provides CR (which Anaxa heavily lacks) and DMG to his primary sources of damage. The SP refund is useful for certain teams, but not a big priority for Cyrene/Cerydra teams. Compared to Anaxa's signature, this LC is of comparable performance without additional DEF shred sources, but Anaxa's LC is stronger with more DEF shred.
The Great Cosmic Enterprise Alternative
Best F2P option with a easily activatable secondary condition (for Anaxa) to get the full DMG boost.

Relic Sets

Genius of Brilliant Stars BiS
General best set for Anaxa since he can implant Quantum weakness onto all enemies. The DEF shred provided from this set also synergises well with his general kit, signature LC and additional DEF shred sources like Cerydra E1.
Scholar Lost in Erudition Alternative
Good and efficient set to farm. Performance is slightly worse than Genius (3% from Genius at E0S1 level), but it provides CR (which Anaxa notably lacks) and DMG to his primary sources of damage.
Pioneer Diver of Dead Waters Alternative
Good set that Anaxa easily satisfies since he inflicts at least 3 debuffs in a fight. Comparable performance to Scholar.
2pc Crit/2pc Crit Alternative
If you don't have any of the above 4pcs.
Eagle of Twilight Line Alternative
Best for sub-DPS Anaxa (w/ SPD boots) and is a potential BiS if you are using Sunday + Cerydra assuming that you get consistent 2-turn ultimates on Anaxa (if Anaxa is on ATK boots).

Planar Ornaments

Not counting E2s, Anaxa S1 = Cerydra E1 = DHPT E1 = Cyrene S1. For light spenders to dolphins, Cyrene E2 provides the best upgrade for value (up to 60%+ in damage output), while the others provide 18-20% damage improvements each.
